Historical Note

Schema for crypto built-in functions in cwm.

These are terms drawn to match the available functions in mxCrypto. See: http://www.amk.ca/python/writing/pycrypt/ for the manual and http://www.amk.ca/python/code/crypto.html for the code.

I am not sure whether in the domain and range functions we should distinguish strings one can present to a person from crypto binary. Might be useful for driving the UI and/or serialization using base64 etc.

Beware that the simplicity of use of some of these properties belies the knowledge o fcrypography that one needs to make a secure system
